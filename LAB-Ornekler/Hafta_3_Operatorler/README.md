# Hafta 3 — Operatörler LAB Örnekleri

Bu dizin, **Bilgisayar Programlama** dersinin 3. haftasına ait uygulamalı C kaynak kodlarını içermektedir.

## Dosya Listesi

| # | Dosya Adı | Konu | Açıklama |
|:-:|:----------|:-----|:---------|
| 1 | `Ornek_1_Aritmetik_Operatorler.c` | Aritmetik Operatörler | Temel aritmetik işlemler: toplama, çıkarma, çarpma, bölme, mod |
| 2 | `Uygulama_1_Operator_Onceligi.c` | Operatör Önceliği | İşlem önceliği ve parantez kullanımının sonuca etkisi |
| 3 | `Uygulama_2_Arttirma_Azaltma.c` | Pre/Post Increment-Decrement | `++` ve `--` operatörlerinin ön-ek ve son-ek farkları |
| 4 | `Uygulama_3_Tip_Donusumleri.c` | Tip Dönüşümleri | Tamsayı bölmesi, implicit/explicit (cast) dönüşümler |
| 5 | `Uygulama_4_Kisa_Devre.c` | Kısa Devre Değerlendirmesi | Mantıksal operatörlerde kısa devre (short-circuit) davranışı |
| 6 | `Uygulama_5_PID_Kontrol.c` | PID Kontrolör Simülasyonu | Operatörlerin mekatronik uygulaması: PID katsayı hesaplama |

## Derleme ve Çalıştırma

```bash
# Tek dosya derleme
gcc -std=c99 -Wall -o cikti dosya_adi.c

# Çalıştırma
./cikti
```
